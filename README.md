# Spam Classifier

Simple Flask-based spam classifier.

Files:
- app.py — Flask app
- train_model.py — example training script
- spam.csv — sample dataset
- model.pkl / vectorizer.pkl — trained artifacts (placeholders)

Quick start:

1. pip install -r requirements.txt
2. python train_model.py  # to train using spam.csv
3. python app.py
